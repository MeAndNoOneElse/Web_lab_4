# Web Lab 4 — Fullstack Web Application

Полноценное веб-приложение с аутентификацией, хранением результатов проверок точек и адаптивным клиентским интерфейсом.

## Описание проекта

Проект состоит из двух основных частей:

- **Backend** на Java (Spring Boot), который предоставляет REST API для аутентификации, управления сессиями и работы с результатами.
- **Frontend** на Angular (папка `front`), который реализует интерфейс пользователя: формы авторизации/регистрации, основную рабочую страницу с графиком, формой ввода и таблицей результатов.

Дополнительно в проекте прорабатывается упрощенный вариант:
- отдельный фронт на Vue (`frontendVUE`)
- отдельный тестовый сервер на JAX-RS без Spring Boot

### Backend

- Spring Boot
- Spring Web (REST)
- Spring Security (JWT)
- Spring Data JPA

### Frontend

- TypeScript

- Многосессионность
- In-memory кеш результатов
- HTTP-фильтры (аутентификация и логирование)
- Разделение на слои: `controller` / `service` / `repository` / `dto` / `entity`

## Что реализовано

### 1) Аутентификация и авторизация
- Регистрация новых пользователей
- Логин с выдачей JWT
- Два типа токенов:
    - **Access Token** (короткоживущий)
    - **Refresh Token** (для обновления access)
- Refresh-token используется с ограничениями безопасности (одноразовое обновление)
- Выход из сессии (logout)

### 2) Управление сессиями
- Поддержка нескольких активных сессий пользователя
- Валидация активной сессии при запросах
- Завершение одной или нескольких сессий
- Периодическая очистка устаревших сессий

### 3) Работа с результатами
- Эндпоинты для добавления и получения результатов проверки точек
- Очистка пользовательских результатов
- Временное кеширование результатов в памяти для ускорения чтения

### 4) HTTP-фильтрация и инфраструктура
- Фильтр проверки JWT и сессии
- Фильтр логирования HTTP-запросов
- Конфигурация CORS и web-настроек

### 5) Клиентская часть
- Экран логина/регистрации
- Главная страница:
    - график
    - форма ввода параметров
    - таблица результатов
- Обработка ошибок и валидация форм
- Адаптивное поведение интерфейса
